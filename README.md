# Higher Level Programming

This repository contains projects from the ALU Higher Level Programming curriculum, focused on Python.

## python-classes

Introduces Object-Oriented Programming (OOP) in Python through the incremental
build-out of a `Square` class:

| File | Description |
|------|--------------|
| `0-square.py` | Empty class `Square` |
| `1-square.py` | `Square` with private instance attribute `size` |
| `2-square.py` | Adds type/value validation for `size` |
| `3-square.py` | Adds `area()` method |
| `4-square.py` | Adds `size` property (getter and setter) |
| `5-square.py` | Adds `my_print()` method |
| `6-square.py` | Adds `position` property for placement when printing |

### Requirements

- All files interpreted/compiled on Ubuntu 20.04 LTS using python3 (version 3.8.5)
- All files should end with a new line
- Code should use the PEP 8 style (version 2.11.*)
- No module imports allowed
