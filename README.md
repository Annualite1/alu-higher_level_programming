# ALU Higher Level Programming

This repository contains projects and exercises completed as part of the
ALU Higher Level Programming curriculum. It covers Python fundamentals,
data structures, object-oriented programming, and more advanced topics.

## Directories

### python-data_structures
Exercises focused on Python lists and basic data structure manipulation,
including printing, formatting, and working with built-in functions like
`str.format()`.

## Requirements

- All scripts are written in Python 3 and tested on Ubuntu (latest version).
- All files start with the shebang line `#!/usr/bin/python3`.
- All files end with a new line.
- Code follows the `pycodestyle` style guide.
- All modules, classes, and functions are documented with docstrings.
- No `import` of external modules unless explicitly allowed.
