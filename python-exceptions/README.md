# Python Exceptions Project

This project contains exercises focused on handling exceptions in Python.

## Learning Objectives

- How to use `try` and `except` blocks
- How to handle errors safely in Python
- How to print elements from lists safely
- Understanding IndexError and other common exceptions

## Files

- 0-safe_print_list.py: Prints elements of a list safely using exception handling

## Requirements

- Python 3
- No external modules allowed
- Must follow PEP8 style guide
