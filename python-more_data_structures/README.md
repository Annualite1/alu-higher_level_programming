# Python - More Data Structures: Set, Dictionary

## Description

This project is part of the ALU Higher Level Programming curriculum. It introduces more advanced Python data structures, with a focus on:

* Sets
* Dictionaries
* Lambda functions
* Map, filter, and reduce
* Data manipulation techniques

Each task is designed to strengthen problem-solving skills while following Python best practices and the project's coding standards.

## Learning Objectives

By the end of this project, I am expected to be able to explain and use:

* Sets and their operations
* Dictionaries and dictionary methods
* Looping through dictionaries and sets
* Lambda functions
* The `map()` function
* The `filter()` function
* The `reduce()` function
* List and dictionary comprehensions

## Requirements

* Ubuntu 20.04 LTS
* Python 3.8.5
* `pycodestyle` 2.7.*
