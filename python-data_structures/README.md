# Python - Data Structures

This project contains Python exercises on data structures as part of the ALU Higher Level Programming curriculum.

## Learning Objectives

- Understand lists and tuples in Python.
- Access, modify, and manipulate list elements.
- Use loops with lists.
- Apply list methods and slicing.
- Write clean, PEP 8 compliant Python code.

## Files

- `0-print_list_integer.py` - Function that prints all integers of a list.
